# Situate Vancouver — frontend

React + TypeScript + Vite insight workspace (MapLibre GL, AI query shell, route tools).

**Project overview, architecture, and run instructions** live in the **[repository root README](../README.md)**.

### Quick commands

```bash
npm install
npm run dev      # Vite dev server (default http://127.0.0.1:5173)
npm test
npm run build
npm run lint
```

- **Insight app:** `/app.html`
- **Marketing landing:** `/` (served from `index.html` at repo root via Vite)
